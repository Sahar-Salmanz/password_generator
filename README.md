# Password Generator
A Password Generator application built using object-oriented design, offering three generation options: \
    - a numeric PIN generator, \
    - a secure random password creator, \
    - and a memorable password generator powered by the Natural Language Toolkit (NLTK).\
The project was created to reinforce key Object-Oriented Programming (OOP) concepts, including structured class design, inheritance, encapsulation, and modular system architecture.


## Features
1. __PIN Code Generator__ 

    - Generates numeric PIN codes of customizable length.
2. __Random Password Generator__

    - Creates secure passwords using a mix of letters, digits, and special characters.

3. __Memorable Password Generator__

    - Uses natural language processing techniques via NLTK library to generate human-readable yet secure passwords based on word combinations.
